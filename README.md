# JsonIt

Simplified JSONIC which is a JSON parser that isn't strict

<!-- [![build status](https://secure.travis-ci.org/yieme/jsonit.png)](http://travis-ci.org/yieme/jsonit) -->

## Installation

This module is installed via npm:

```sh
npm i jsonit -g
```

## Example Usage

```js
jsonit --help
```
